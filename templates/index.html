Submit a clickstream event

Use this form to manually submit a real user event to the database, or use the API endpoint POST /api/ingest from your website's JS tracker.

JS tracker snippet

Drop this into your website to track real user events: 

async function trackEvent(page, action, timeSpent) {

  await fetch('http://localhost:5000/api/ingest', {

    method: 'POST',

    headers: {
      'Content-Type': 'application/json'
    },

    body: JSON.stringify({

      user_id:
        localStorage.getItem('uid') || 'guest',

      session_id:
        sessionStorage.getItem('sid') || Date.now(),

      page,
      action,
      time_spent: timeSpent
    })
  });
}
